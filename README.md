# Kritpi Portal

My personal portfolio website, built with Next.js, and Tailwind CSS.

The site has two modes:

<!-- - Portfolio mode: a clean black-and-white portfolio view with About me, Work Experience, Projects, Education, Skills, and Soft Skills. -->
- Portfolio mode: a clean, friendly portfolio view with some of my experiences and information
- CV mode: a formal, print-friendly CV layout.

## Mode Switch

Toggle in the top-right corner to change between Portfolio and CV mode.

## Quick Installation

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

## Available Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```
